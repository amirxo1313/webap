# 🎵 Behimelobot - Complete Project Code

## نحوه استفاده:
1. این فایل را کپی کنید
2. در سرور خود پوشه‌ای به نام behimelobot بسازید
3. فایل‌ها را طبق ساختار زیر ایجاد کنید

## دستورات نصب:
```bash
mkdir behimelobot
cd behimelobot
npm init -y
# سپس فایل‌ها را کپی کنید
npm install
npm run build
npm start
```
